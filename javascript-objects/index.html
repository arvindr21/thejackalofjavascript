{ Object }

The single entity that the Object Oriented World revolves around!!

What is an Object?

An object is anything that occupies space! In the world of Computer Science a.k.a co.si., It is a location in the memory that stores something. Something being a value or another object or an address to another object.

Narrowing down to Javascript objects, these are a collection of properties and each property is a key value pair. And a value can be another function or variable.

So, in the real world, how can we relate to objects ?

A building is an object, it has properties like floors, terrace, etc… Each floor can be an object. Each floor has rooms, elevators, hallways. Each room  can be an object and it has properties like walls, doors, windows. And each wall can be an object where a wall clock, a painting or a light saber (on the wall) can be an object.

Easy to relate right? Its the same in the programming world too. Lets take an example of a car.

Car is an Object – It has properties like name, brand, type, cost, color etc.

Here the “car” represents the base entity for a garage/fleet of cars. Typically, the any car inside the garage/fleet is a type of car. i.e. Some way or the other they exhibit the same properties. The values may differ, but the properties are the same.

So now, lets say you want to see all the properties inside a given object. How would you go about it?

There are  3 ways

  1. for..in
  2. Object.keys(obj);
  3. Object.getOwnPropertyNames(obj);
